# IsekaiBattle
My first game using Java

# Warning
Please Don't choose to Play in 1 VS 1 since it won't work
This was an attempt to implement a 1 vs 1 between two players but I got stuck, so choose Single Player mode and fight against a primitive AI

# How To Play
N: for normal attacks\n
F: for final attack only if the blue bar is full;
S: for special attack only if the blue bar is half full or more;
T: to transform;
E: to escape from the enemy;
Up: to jump;
Left and Right: move;
